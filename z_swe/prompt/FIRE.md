# FIRE: Financial Independence, Retire Early

## Turn 1: Setup Static JavaScript web application for FIRE
Goal:
	- Create a static web application that helps users track their progress towards financial independence, retire early. The implementation locates in z_swe/FIRE folder (empty currently)

Implementation details:
	1. Use Node.js, express, ReactJS, TailwindCSS as a static web application.
	2. The application should have the following features:
	- Users can view their progress towards financial independence, retire early.
	- Users can input their current information (current age, current savings, monthly income, monthly expenses)
		and calculate the estimated time to reach financial independence, retire early.
	3. The UI must be modern, use orange variant as primary color and white as background color, 
		simply, clearly and beautifully.
	4. The UX must be smooth, fast and responsive.
	5. Create README.md that provide setup steps
